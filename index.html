Letter Match (React Web App)

Interactive React-based web application featuring timed gameplay and dynamic UI components. Contributed to frontend structure, state management, and user interaction features within a team-built project.

MY ROLE: I worked on the Letter Match Game while my teammates worked on the other games, like Word Chain and Word Blitz. The modes include single player, same device multiplayer, and online mode. In this game, you are assigned a letter like "B" and you have to write a name, person, place, thing, and animal starting with that letter. You need to enter as many fields as you can before time runs out, and each category is worth 10 points. I also matched the words to each letter and category, along with points validation.
